## Principio Rector

Toda decision prioriza, en este orden:

1. Reducir riesgo para pacientes y operacion.
2. Resolver la necesidad con la menor complejidad posible.
3. Usar software libre o planes gratuitos sostenibles.
4. Documentar costos ocultos de infraestructura, mantenimiento y soporte.
5. Pagar solo cuando una necesidad y un beneficio medible lo justifiquen.
