    `npm run audit`, `npm test`, `npm run build`. Anadir a package.json un script `verify`
    que encadene `npm run audit && npm test && npm run build` para invocacion unica desde CI
    y local. Este pipeline se extendera en planes posteriores (e2e, axe, lighthouse,
    validate-content, check-hardcoded).
  </action>

  </verify>

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-03 | Tampering | acciones de terceros mutables | mitigate | fijar todas las acciones por SHA de commit completo |
