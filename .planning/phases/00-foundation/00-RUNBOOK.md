# Runbook Inicial De Fase 0 Y MVP

## Incidente De Contenido Incorrecto

1. Retirar o despublicar el contenido afectado.
2. Registrar pagina, version, hora y quien detecto el problema.
3. Notificar a la doctora.
4. Corregir solo con aprobacion.
5. Verificar despliegue y registrar cierre.

## Compromiso De Cuenta

1. Revocar sesiones y credenciales afectadas.
2. Rotar secretos relacionados.
3. Revisar historial de cambios y despliegues.
4. Restaurar ultima version aprobada.
5. Documentar impacto y acciones preventivas.

## Caida Del Sitio

1. Confirmar si el proveedor, dominio o despliegue fallo.
2. Mantener accesible el canal alternativo fuera del sitio cuando sea posible.
3. Restaurar el ultimo despliegue aprobado.
4. Registrar duracion, causa y correccion.

## Recepcion De Informacion Clinica

1. No copiarla a herramientas adicionales.
2. Informar que el canal no reemplaza atencion medica.
3. Aplicar el procedimiento aprobado por la doctora.
4. Escalar urgencias segun mensaje aprobado, sin diagnosticar.
5. Registrar solo el minimo administrativo permitido.

## Rollback Del Sitio

1. Identificar ultima version aprobada.
2. Restaurar mediante el mecanismo del proveedor o redesplegar desde Git.
3. Ejecutar pruebas de humo.
4. Confirmar canales de contacto.
5. Registrar evidencia.
